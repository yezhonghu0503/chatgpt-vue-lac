<p>vue3和vue2使用marked的方式基本一致，只不过vue3的setup语法要进行微调。</p> <ol> <li><p>在vue项目中安装marked：<code>npm install marked</code></p> </li> <li><p>在需要使用marked的组件中引入marked：<code>import marked from &#39;marked&#39;</code></p> </li> <li><p>在组件的setup方法中，使用marked将markdown文本转化为html：</p> </li> </ol> <pre><code class="language-javascript">import marked from &#39;marked&#39; export default { name: &#39;Markdown&#39;, props: [&#39;text&#39;], setup(props) { const html = marked(props.text) return { html } } } </code></pre> <ol start="4"> <li>在模板中使用html渲染markdown内容：</li> </ol> <pre><code class="language-html">&lt;template&gt; &lt;div v-html=&quot;html&quot;&gt;&lt;/div&gt; &lt;/template&gt; </code></pre>
