vue3和vue2使用marked的方式基本一致，只不过vue3的setup语法要进行微调。 1. 在vue项目中安装marked：`npm install marked` 2. 在需要使用marked的组件中引入marked：`import marked from 'marked'` 3. 在组件的setup方法中，使用marked将markdown文本转化为html： ```javascript import marked from 'marked' export default { name: 'Markdown', props: ['text'], setup(props) { const html = marked(props.text) return { html } } } ``` 4. 在模板中使用html渲染markdown内容： ```html <template> <div v-html="html"></div> </template> ```
